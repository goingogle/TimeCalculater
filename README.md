# TimeCalculater
这是一个我自建的网页小工具，只有那个跟计算器一样的时间计算器好好用过，用来算乙方单位设备调拨时间的。This is a web-based tool I built myself. The only feature that has been truly useful is the time calculator, which functions like a regular calculator and is used to calculate equipment transfer times for Party B's units.

基于 Vue 3 + TypeScript + Vite 构建。
